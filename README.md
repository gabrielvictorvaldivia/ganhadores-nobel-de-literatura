# Prêmio Nobel de Literatura (1901-2024)

Este repositório contém dados completos sobre os ganhadores do Prêmio Nobel de Literatura desde sua criação em 1901 até 2024.

## Sobre o Dataset

O dataset inclui informações detalhadas sobre cada laureado, incluindo:
- **Ano**: Ano em que o prêmio foi concedido
- **Laureado**: Nome do ganhador
- **País**: País de origem
- **Aceitação**: Se o prêmio foi aceito ou não
- **Atividade Principal**: Área de atuação (poeta, romancista, dramaturgo, etc.)
- **Obra Específica Premiada**: Trabalho pelo qual foi reconhecido
- **Idioma Original**: Língua em que a obra foi escrita
- **Gênero Principal**: Categoria literária
- **Justificativa Oficial**: Razão oficial para a concessão do prêmio
- **Outras Obras Principais**: Trabalhos adicionais relevantes
